## 指定ping包大小

```shell
ping 192.168.0.173 -l 1024 -n 2 //2 发送两次
hping3 -S -d 1024 192.168.4.4 
hping3 -c 1000 -d 5000 -S --flood 192.168.4.3 //发送1000个包，每个包大小为5000
```

## 修改虚拟机ip信息

```shell

vi ifcfg-ens32  
service network restart  //重启 输入ip addr 查看是否修改正确
arp -a //查看局域网内所有的ip
```
